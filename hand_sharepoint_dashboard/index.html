About this dashboard
Purpose, what is included, and key definitions.

This dashboard is a public-facing mapping snapshot of informal settlements in Lebanon. It is designed for dissemination: quick KPIs, interactive map exploration, and exports.

Definitions
  • Assessed: a record with a phone assessment status recorded.
  • Active site: site status equals Active.
  • QC flags: automatic indicators for missing or inconsistent fields that may need review.
Disclaimers
  • This public build excludes personal contact details.
  • Boundaries and names are shown for reference and do not imply official endorsement.
